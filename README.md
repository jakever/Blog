# vue-blog

> 使用vue2、koa2、mongodb搭建的个人博客，支持markdown编辑，文章标签分类，发布文章／撤回发布文章，支持服务端渲染（Server-Side Rendering）

## Build Prerequisite
1. 安装mongoose
2. 确保NodeJs版本是6.0以上


## Build Setup

``` bash
# install dependencies
npm install

# 启动客户端
npm run dev-client

# 启动服务端
npm run dev-server

# build for production with minification
npm run build
```
